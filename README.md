Gene Ontology Graph Analysis – Homo sapiens

Overview

This project provides a Python library for loading, representing, and analyzing the Gene Ontology (GO) and its annotations for Homo sapiens. The Gene Ontology is modeled as a directed acyclic graph (DAG), and Gene Ontology Annotations (GOA) are integrated as links between GeneProducts and GO terms.

The project combines graph algorithms, data structures, and biological interpretation to study the structure of the GO and the functional annotation of GeneProducts.

Objectives

The main objectives of this project are:

- Load and represent the Gene Ontology structure from OBO files
- Load Gene Ontology annotations from GOA/GAF files
- Navigate the GO hierarchy (parents, children, ancestors, descendants)
- Query relationships between GeneProducts and GO terms
- Analyze the structure of the ontology (depth, longest paths)
- Compute biological statistics related to annotations

Data Sources

Gene Ontology structure (OBO)

The OBO file contains the complete definition of GO terms, including identifiers, names, namespaces, definitions, hierarchical relations (is_a, part_of), and alternative identifiers. This file is used to reconstruct the directed acyclic graph of the Gene Ontology.

Download link (official):
https://current.geneontology.org/ontology/go-basic.obo

Version information:
- Format version: 1.2
- Release date: 2025-10-10

Gene Ontology Annotations (GOA)

The GOA file provides associations between GeneProducts and GO terms, along with metadata such as GeneProduct identifiers, names, evidence codes, annotation dates, and source databases.

Download link (official, UniProt):
https://ftp.ebi.ac.uk/pub/databases/GO/goa/proteomes/25.H_sapiens.goa

Version information:
- GOA format version: 2.2
- Generated by UniProt: 2025-11-10
- Last update: 2025-12-02

Installation

This project requires Python 3.8 or higher. To set up the environment, follow these steps:

Clone the repository and move to the project directory:
```{bash}
git clone https://github.com/bachri31/GeneOntology
cd GeneOntology
```
No needs to install external packages; only standard libraries are used.

R is only required to reproduce the figures and statistical analyses included in the report.

Usage

Load the Gene Ontology and annotations
```{python}
from src.geneontology import load_OBO, load_GOA

go = load_OBO("data/go-basic.obo")
load_GOA(go, "data/25.H_sapiens.goa")
```

Retrieve GeneProducts associated with a GO term

```{python}
from src.geneontology import GeneProducts

gps_direct = GeneProducts(go, "GO:0006915", recursive=False)
gps_recursive = GeneProducts(go, "GO:0006915", recursive=True)
```

Retrieve GO terms associated with a GeneProduct

```{python}
from src.geneontology import GOTerms

terms_direct = GOTerms(go, "P04637", recursive=False)
terms_recursive = GOTerms(go, "P04637", recursive=True)
```

Compute maximal depth of the Gene Ontology

```{python}
from src.geneontology import max_depth_go
depth_bp = max_depth_go(go, namespace="biological_process")

depth, deepest_term, path = max_depth_go(
    go,
    namespace="biological_process",
    return_path=True
)
```

Analyses Performed

The project includes the following analyses:
- Number of nodes and edges in the GO graph
- Distribution of GO terms across ontologies (Biological Process, Molecular Function, Cellular Component)
- Maximal depth of each ontology
- Distribution of GeneProducts per GO term
- Distribution of GO annotations per GeneProduct
- Distribution of evidence codes

Results are exported as CSV files and visualized using R and ggplot2.
